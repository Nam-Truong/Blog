# Blog

## Project setup

1. System requirements:
  
    - nodejs: v10.24.1 (preferred)

2. Development:

    ```
    npm run build
    npm run dev
    ```

    **OR**

    ```
    npm run build
    npm run start
    ```

3. Deployment:

    Create static HTML/CSS files inside directory _./docs_
    ```
    npm run generate
    ```

    **Note**: Do Not Remove The File _./docs/CNAME_
